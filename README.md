## Overview

*Somnium* is a retro-inspired first-person dungeon crawler.  
The game was designed with accessibility at its core, with all major systems — including combat, loot boxes, navigation, and UI — built to support a wide range of players.

The project was developed as part of a collaborative team production and later received:

🏆 **Best Debut Game – Futuregames Awards**

---

## Main Responsibilities

### Enemies & AI
- Implemented drag-and-drop enemy spawn prefabs  
- Designed aggro and attack behaviors  
- Integrated Unity NavMesh for enemy navigation  
- Created dynamic spawn timer systems  

### User Interface
- Built interactable world UI prompts  
- Implemented player HUD (health, upgrades, controller tutorial)  
- Designed feedback systems for combat clarity  

### Interactables
- Developed interaction system for static world objects  
- Implemented potions, upgrades, loot boxes, and weapons  
- Created a quest-driven world state change system tied to object pickup events  

### Accessibility Prototyping
- Prototyped a breadcrumb navigation system inspired by *God of War Ragnarök*  
- Contributed to alternative accessibility-driven guidance systems  

---

## Core Gameplay Features

### Accessible Combat System
- Auto-aim assist for players with motor impairments  
- Upgradeable homing projectiles  
- Directional damage indicator overlay  

### Accessibility Options
- Adjustable pixel filter  
- Adjustable brightness  
- Screen shake toggle  
- Camera bob toggle
